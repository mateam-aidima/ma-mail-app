⚙️ 初期設定

Google Cloud ConsoleでOAuth 2.0クライアントIDを作成し、以下を入力してください。

Google Cloud Console → APIとサービス → 認証情報
📋 Google Cloud Console 設定手順
1. Google Cloud Console にアクセス
2. プロジェクトを作成し Gmail API を有効化
3. 認証情報 → OAuth 2.0 クライアントID（ウェブアプリ）を作成
4. 承認済みJavaScriptオリジン に http://localhost:8888 を追加
5. クライアントIDをここに貼り付け

📁 新規PJ作成

「PJ-」から始まる番号を入力してください
